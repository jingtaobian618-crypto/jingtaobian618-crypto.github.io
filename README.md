## 📢 核心政治主张
* **主权在民**：确立一人一票普选权。
* **财产神圣**：确立土地私有权，保护私人财产。
* **司法独立**：引入公民陪审团制度。
* **民生尊严**：全民免费医疗、义务教育与养老保障。
