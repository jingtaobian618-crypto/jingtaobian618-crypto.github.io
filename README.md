## 📢 我们的核心愿景 (WHO WE ARE)
我们不是权力的争夺者，而是宪政的修路人。

### 1. 财产与土地私有
彻底废除土地公有制，确立公民对土地及私有财产的永久所有权。让“有恒产者有恒心”。

### 2. 司法公正与陪审团
引入公民陪审团制度。让普通国民直接参与裁决，确立“司法权属于人民”的最终底线。

### 3. 全民福利底线
国家财政优先保障：十二年免费义务教育、全民免费医疗、统一标准的国民基础养老金。

### 4. 绝对透明的政府
基于区块链技术公示政府每一笔财政开支。所有公职人员财产必须全网实时公示。
